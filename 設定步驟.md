# 穿搭小助手 LINE Bot 設定指南

每晚 22:00 自動推播明日穿搭建議到你的 LINE！

---

## 需要準備

- LINE 帳號
- GitHub 帳號（用來連接 Railway）
- Google API Key（免費，[申請網址](https://aistudio.google.com)）

---

## 第一步：建立 LINE Bot 頻道

1. 前往 **[LINE Developers](https://developers.line.biz)**，用 LINE 帳號登入

2. 左側選單 → **Create** → **Create a new provider**
   - 名稱填：`穿搭小助手`（或任意名稱）→ **Create**

3. 在 Provider 頁面 → **Create a new channel** → 選 **Messaging API**

4. 填寫資訊：
   - Channel type: `Messaging API`
   - Provider: 剛建立的 provider
   - Company or owner's country: `Taiwan`
   - Channel name: `穿搭小助手`
   - Channel description: `每日 AI 穿搭建議`
   - Category / Subcategory: 任選
   - Email: 你的 email
   - 勾選同意條款 → **Create**

5. 建立完成後，進入頻道：
   - **Basic settings** → 捲到最下方 → 找到 **Channel secret** → 複製並記下

6. 切換到 **Messaging API** 頁籤：
   - 捲到最下方 → **Channel access token** → 點 **Issue** → 複製並記下

7. 同頁面找到 **Auto-reply messages** → 點 **Edit** → 關閉（Disabled）

8. 找到 **Greeting messages** → 同樣關閉（Disabled）

---

## 第二步：上傳程式碼到 GitHub

1. 前往 [github.com](https://github.com) 登入 → **New repository**
   - Repository name: `outfit-line-bot`
   - 設為 **Private**
   - **Create repository**

2. 在你的電腦開啟命令提示字元（cmd），執行：

```
cd "C:\Users\sharo\OneDrive\claude\穿搭小助手\line-bot"
git init
git add .
git commit -m "初始版本：穿搭小助手 LINE Bot"
git remote add origin https://github.com/你的帳號名/outfit-line-bot.git
git push -u origin main
```

> ⚠️ 把 `你的帳號名` 改成你的 GitHub 帳號

---

## 第三步：部署到 Railway

1. 前往 **[railway.app](https://railway.app)** → 用 GitHub 帳號登入

2. 點 **New Project** → **Deploy from GitHub repo**

3. 選擇剛建立的 `outfit-line-bot` repo → **Deploy Now**

4. 等待部署完成（約 1-2 分鐘）

5. 部署完成後，點 **Settings** → **Domains** → **Generate Domain**
   - 你會得到一個網址，例如：`https://outfit-line-bot-xxxx.railway.app`
   - **複製這個網址**

6. 點 **Variables** → 新增以下環境變數：

| Key | Value |
|-----|-------|
| `LINE_CHANNEL_SECRET` | 第一步複製的 Channel secret |
| `LINE_CHANNEL_ACCESS_TOKEN` | 第一步複製的 Channel access token |
| `GOOGLE_API_KEY` | 你的 Google API Key（格式：AIzaSy...）|
| `TZ` | `Asia/Taipei` |

7. 新增變數後 Railway 會自動重新部署

---

## 第四步：設定 LINE Webhook

1. 回到 LINE Developers → 你的頻道 → **Messaging API** 頁籤

2. 找到 **Webhook URL** → 點 **Edit**

3. 填入：`https://你的railway網址/webhook`
   - 例如：`https://outfit-line-bot-xxxx.railway.app/webhook`

4. 點 **Update** → 再點 **Verify**（應顯示 "Success"）

5. 把 **Use webhook** 切換為 **Enabled**

---

## 第五步：加 LINE Bot 為好友並使用

1. 在 LINE Developers → **Messaging API** → 找 **QR code**
2. 用手機掃碼加好友
3. 傳送衣物照片給 Bot → Bot 會儲存到衣櫃
4. 傳「今日穿搭」→ 立即獲得 AI 建議
5. 每晚 **22:00** 自動收到明日穿搭推播！

---

## 指令清單

| 傳送 | Bot 回覆 |
|------|---------|
| 衣物照片 | 加入衣櫃，顯示目前件數 |
| `今日穿搭` | AI 分析衣櫃，生成建議 |
| `今日穿搭 婚禮` | 針對特定場合生成建議 |
| `衣櫃` | 顯示目前衣物件數 |
| `清空衣櫃` | 清除所有衣物照片 |

## 常見問題

**Q: Webhook Verify 失敗？**
確認 Railway 部署完成且 Domain 已生成，網址後面要加 `/webhook`

**Q: Bot 沒有回覆？**
確認環境變數設定正確，到 Railway → Deployments → 查看 Logs

**Q: 衣物照片消失了？**
Railway 重新部署時 JSON 檔案會重置，重新傳送照片即可（未來可升級到有永久儲存的方案）

**Q: 如何更換推播時間？**
修改 `server.js` 第 95 行的 cron 時間，重新 push 到 GitHub 即可自動部署
